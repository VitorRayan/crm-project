# CRM Project

Um sistema de CRM (Customer Relationship Management) moderno e responsivo desenvolvido para facilitar o gerenciamento de clientes, acessos e status.

## Funcionalidades

- **Dashboard Interativo:** Interface moderna e intuitiva para gerenciamento de clientes.
- **Gerenciamento de Clientes (CRUD):** Criação, edição, exclusão e leitura de informações de clientes.
- **Busca Rápida:** Filtro em tempo real de clientes por nome ou e-mail.
- **Status Visual:** Indicação em tempo real de status (Ativo/Inativo) com badges e animações.
- **UI/UX Moderna:** Design limpo focado na experiência do usuário, totalmente responsivo e utilizando variáveis de CSS para padronização (Design Tokens).

## Tecnologias Utilizadas

- **Frontend:** React, Vanilla CSS, Axios, React Icons.
- **Backend:** Node.js, Express.

## Como Executar o Projeto

### Pré-requisitos
Certifique-se de ter o **Node.js** e o **npm** instalados na sua máquina.

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/VitorRayan/crm-project.git
   cd crm-project
   ```

2. **Inicie o Backend:**
   ```bash
   cd backend
   npm install
   npm run start:dev
   ```
   *(A API REST iniciará na porta 3000)*

3. **Inicie o Frontend:**
   Em um novo terminal, abra a pasta do frontend e execute:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. **Acesse a aplicação:**
   Abra seu navegador no endereço indicado pelo terminal do frontend.
